This repository contains my **Week 1 C++ Assignment** for the Basics of C++ course.

## 📚 Topics Covered
- Syntax
- Variables
- Data Types
- Input/Output
- Operators
- Loops
- Conditional Statements

## 📝 Assignments

### 1. Simple Calculator
A program that performs:
- Addition
- Subtraction
- Multiplication
- Division

### 2. Sum and Average of Array Elements
A program that:
- Takes array input
- Calculates the sum
- Calculates the average

### 3. Prime Number Checker
A program that checks whether a given number is prime or not.

### 4. Reverse a Number
A program that reverses the digits of a given number.

## 💳 Mini Project – ATM Menu Program

Features:
- Check Balance
- Deposit Money
- Withdraw Money
- Exit

## ▶️ How to Run

1. Compile the program using any C++ compiler.
2. Run the executable.
3. Select an option from the menu.
4. Follow the on-screen instructions.
